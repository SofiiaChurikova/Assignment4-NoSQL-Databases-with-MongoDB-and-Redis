# NoSQL Databases Assignment — MongoDB and Redis

## Contents

- `mongodb_assignment.js` — all MongoDB commands
- `mongodb_optimization_report.pdf` — query optimization report
- `redis_assignment.txt` — all Redis commands
- `redis_notes.md` — short notes on Redis use cases
- `screenshots/` — screenshots of the commands and their results
- `README.md`

## Tools used

- DataGrip
- MongoDB Atlas — collection `books`
- Redis Cloud — used as a cache layer

## How to run

**MongoDB:** open a MongoDB console in DataGrip and run the blocks in `mongodb_assignment.js` in order .

**Redis:** open a Redis console in DataGrip and run the commands in
`redis_assignment.txt` one per line.
